## hand
The aim in this project is to build a robotic hand that can be controlled by the brain signals sent to the muscles in the forearm. The later is achieved by using Surface Electromyography (sEMG) to read those signals. After, I plotted the data in real time and used a classifier to identify specific finger movement. This become useful as a hand prosthesis. The fingers still on the works.
